How to start with the GUI ?

To develop a dynamic and intuitive graphical user interface, we used libraries. Normally, some of them are include in Python distribution. You can find below the list of libraries which are used to create the UI.

List of libraries for the GUI

Modules which are bundled in the Python installation :

Tkinter re sys decimal

Modules which need to be installed in addition to Python :

NumPy & SciPy Matplotlib Pillow (PIL)

Install modules to Python 2.X

sudo apt-get install python-tk
pip -m install Pillow
python -mpip install -U pip
python -mpip install -U matplotlib
pip install -U numpy scipy

Install modules to Python 3.X

sudo apt-get install python3-tk
pip -m install Pillow
python -mpip install -U pip
python -mpip install -U matplotlib
pip install -U numpy scipy
